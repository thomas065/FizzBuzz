The Code is structured in three functions.

getValues()

getValues obtain's the user's input from Fizz and Buzz value which is then accessed by the getElementById variable.

fizzBuzz(fizzValue, buzzValue)

This function consists of two arguments fizzValue and buzzValue. We check if the number from the user input is an integer first at which point if true, then it will be passed in to the next function.

displayData(fbArray)

Once the preceding integers are parsed and checked by an If/Else statement, displayData begins the for loop process to finally give back the user's requested data.
